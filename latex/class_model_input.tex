\doxysubsection{Detailed Description}
Manages frame preprocessing for emotion detection analysis. 

The \doxylink{class_model_input}{Model\+Input} class handles the preprocessing of input frames for emotion detection analysis. It provides methods to process input images, set and retrieve frames, and manage regions of interest.
