std::vector< std::string > EmotionDetection::emotionANL ( ModelInput & image)

Function to analyze the emotion.

This function gets the pre-processed image and converts it into a blob which can be fed into the neural network. The classID with the highest probability is determined, and the corresponding emotion label is added to a vector.

Parameters
imageThe pre-processed image which will be analyzed to determine the emotions of the person in the image.
Returns
std::vector<std::string> The vector of emotion labels predicted for the image.
